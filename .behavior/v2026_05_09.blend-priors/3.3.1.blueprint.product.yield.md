# blueprint: sdk-aws-lambda

> merge simple-lambda-handlers + simple-lambda-client into unified SDK

---

## summary

build a unified aws lambda SDK that provides:

1. **genLambdaEndpoint** — create type-safe lambda handlers with zod validation and trail context
2. **genLambdaEndpoint.forApiGateway** — api gateway handler with cors, security headers, body parse
3. **askLambdaEndpoint** — invoke lambdas with trail propagation and cache dedup

key modernizations:
- zod v4 (replaces joi)
- @aws-sdk/client-lambda v3 (replaces aws-sdk v2)
- middy v7 (replaces middy v1)
- node 18+ required

## codepath tree

### genLambdaEndpoint

```
genLambdaEndpoint({ schema, invoke, logTranslate? }, context?)
├── [+] createTrailMiddleware
│   ├── extract trail.exid from event (if present)
│   ├── generate exid if absent
│   └── inject trail into context.log
│
├── [+] createIoLogMiddleware (before)
│   ├── log.debug('handler.input', logTranslate.input(event))
│   └── logTranslate enables redaction (auth headers, PII, large payloads)
│
├── [+] createValidationMiddleware
│   ├── validate event against schema.input
│   └── throw BadRequestError on failure
│
├── [+] invoke user function
│   ├── pass { with: validatedInput } as input
│   └── pass { log: trailAwareLog } as context
│
├── [+] createOutputValidationMiddleware
│   ├── validate result against schema.output
│   └── throw InternalServiceError on failure
│
├── [+] createIoLogMiddleware (after)
│   └── log.debug('handler.output', logTranslate.output(result))
│
├── [+] createIoLogMiddleware (onError)
│   ├── if response exists: log.debug('handler.output', logTranslate.output(response))
│   └── if no response: log.debug('handler.output', { errorMessage, stackTrace })
│
└── [+] return result
```

### genLambdaEndpoint.forApiGateway

```
genLambdaEndpoint.forApiGateway({ schema, invoke, cors, deserialize? })
├── [←] @middy/http-json-body-parser (if deserialize.body !== false)
│   └── JSON.parse(event.body)
│   └── skip if deserialize.body: false (e.g., webhook signature verification)
│
├── [←] @middy/http-cors
│   ├── handle OPTIONS preflight → 204
│   └── add cors headers to response
│
├── [←] @middy/http-security-headers
│   ├── X-Content-Type-Options: nosniff
│   ├── X-Frame-Options: DENY
│   └── X-XSS-Protection: 1; mode=block
│
├── [○] (reuse genLambdaEndpoint middleware chain)
│   ├── trail middleware
│   ├── validation middleware
│   ├── invoke
│   └── output validation
│
├── [+] createErrorHandlerMiddleware
│   ├── catch errors
│   ├── decideIsBadRequestError (thorough: instanceof, name, AND prototype chain)
│   ├── BadRequestError → 400 + { errorMessage, errorType, causeMessage? }
│   ├── InternalServiceError → 500 + sanitized (no stack traces)
│   └── include error.cause chain if present
│
├── [←] @middy/http-response-serializer
│   └── JSON.stringify(body) for application/json
│
└── [+] format response
    └── { statusCode, headers, multiValueHeaders?, body }
```

### askLambdaEndpoint

```
askLambdaEndpoint<TReq, TRes>({ which, event }, context?)
├── [+] asLambdaFunctionName
│   └── `${service}-${stage}-${function}`
│
├── [+] extract trail
│   ├── from context.log.trail.exid if present
│   └── generate if absent
│
├── [+] asLambdaPayload
│   └── { ...with, trail: { exid } }
│
├── [←] cache check (reuse with-simple-cache)
│   └── return cached if hit
│
├── [+] invoke lambda
│   ├── new LambdaClient()
│   └── client.send(new InvokeCommand({ FunctionName, Payload }))
│
├── [+] asParsedResponse
│   ├── parse Payload as JSON
│   ├── detect error (errorMessage, errorType, stackTrace)
│   └── throw LambdaInvocationError if error
│
├── [+] cache store
│   └── store success in cache
│
└── [+] return typed result
```

### acceptance test harness

in-process harness enables e2e journey tests without AWS credentials:

```ts
// __test_assets__/createInProcessLambdaHarness.ts
export const createInProcessLambdaHarness = (handlers: Record<string, Record<string, Handler>>) => {
  jest.mock('@aws-sdk/client-lambda', () => ({
    LambdaClient: jest.fn().mockImplementation(() => ({
      send: jest.fn().mockImplementation(async (command) => {
        const fnName = command.input.FunctionName;
        const [service, fn] = fnName.split('-').slice(0, 2).join('-').split('-');
        const handler = handlers[service]?.[fn];
        if (!handler) throw new Error(`no handler: ${fnName}`);
        const event = JSON.parse(command.input.Payload);
        const result = await handler(event, {});
        return { Payload: Buffer.from(JSON.stringify(result)) };
      }),
    })),
  }));
};
```

this enables:
- **hermetic**: no network, no AWS credentials
- **fast**: runs in milliseconds, parallelizable
- **real serialization**: JSON.stringify/parse through the chain
- **real handler**: genLambdaEndpoint receives actual event shape

## cors defaults

delegates to @middy/http-cors. uses middy's options type directly (no wrapper).

default Access-Control-Allow-Headers:
- Content-Type
- Authorization
- X-Request-Id
- X-Amz-Date
- X-Api-Key
- X-Amz-Security-Token

user can override via `cors` config passed to genLambdaEndpoint.forApiGateway.
