# blueprint: introspection for lambda endpoints

## summary

add runtime schema introspection to lambda endpoints:
1. handler-side: middleware intercepts `{ introspect: 'schema' }` payloads and returns JSON schema (prep-only)
2. caller-side: `getOneLambdaContract` and `getAllLambdaContracts` functions to query endpoint schemas

## codepath tree

### handler-side (introspection middleware)

introspection middleware is shared between both endpoint types:

```
genLambdaEndpoint.forAskEndpoint (entry)
├── [○] genConstraintErrorMiddleware     # retain: error handler
├── [○] genInternalServiceErrorMiddleware # retain: error handler
├── [○] genIoLoggerMiddleware            # retain: io logger
├── [○] genTrailMiddleware               # retain: trail injection
├── [+] genIntrospectionMiddleware                   # CREATE: introspection intercept
│   ├── [+] genIntrospectionMiddleware.isIntrospectionPayload  # detect { introspect: 'schema' }
│   ├── [+] genIntrospectionMiddleware.getJsonSchemaFromZod    # convert zod → JSON schema
│   └── [←] request.response pattern                 # REUSE from genConstraintErrorMiddleware
├── [○] genZodEventValidationMiddleware  # retain: input validation
└── [○] invoke                           # retain: handler logic

genLambdaEndpoint.forApiGateway (entry)
├── [○] genConstraintErrorMiddleware     # retain: error handler (apiGateway: true)
├── [○] genInternalServiceErrorMiddleware # retain: error handler (apiGateway: true)
├── [○] genIoLoggerMiddleware            # retain: io logger
├── [○] httpResponseSerializer           # retain: response serialization
├── [○] httpCors                         # retain: CORS headers
├── [○] httpSecurityHeaders              # retain: security headers
├── [○] genApiGatewayEventNormalizationMiddleware # retain: event normalization
├── [○] genTrailMiddleware               # retain: trail injection
├── [+] genIntrospectionMiddleware                   # CREATE: introspection intercept (apiGateway: true)
│   ├── [+] genIntrospectionMiddleware.isIntrospectionPayload  # detect { introspect: 'schema' } in body
│   ├── [+] genIntrospectionMiddleware.getJsonSchemaFromZod    # convert zod → JSON schema
│   └── [←] request.response pattern                 # returns { statusCode: 200, body: schema }
├── [○] genZodBodyValidationMiddleware   # retain: body validation
└── [○] invoke                           # retain: handler logic
```

note: for API Gateway, introspection payload arrives in `event.body` and response is wrapped in `{ statusCode: 200, body: LambdaEndpointSchema }`.

### caller-side (contract query)

```
getOneLambdaContract (entry)
├── [←] getLambdaFunctionName            # REUSE: build function name
├── [inline] { introspect: 'schema' }    # inline payload (one-liner)
├── [←] genLambdaClient                  # REUSE: get lambda client (extant code)
├── [←] executeLambdaInvocation          # REUSE: invoke lambda
└── [+] asParsedIntrospectionResponse    # CREATE: parse schema response

getAllLambdaContracts (entry)
├── [+] getAllLambdaFunctionsByPrefix    # CREATE: communicator for ListFunctionsCommand
├── [←] getOneLambdaContract             # REUSE: get each endpoint's schema
└── [+] asContractRecord                 # CREATE: aggregate array into Record<string, Schema>
```

note: `genLambdaClient` uses "client" term — this is extant code. rename to "caller" is out of scope for this feature (separate refactor ticket).

note: `genLambdaEndpoint.forAskEndpoint` uses `for*` pattern — this is extant code. the pattern is out of scope for this feature.

note: transformers use `as*` and `is*` prefixes per rule.require.named-transformers.md — these are not orchestrators, so get/set/gen verbs do not apply. `asContractRecord`, `asParsedIntrospectionResponse`, and `isIntrospectionPayload` are all transformers.

    when('[t0] all contracts queried by service prefix', () => {
      const contracts = useThen('all schemas returned', async () =>
        getAllLambdaContracts(
          { which: { service: 'svc-test' } },
          { ...context, env: { access: 'prep' } },
        ),
      );

        expect(contracts['getCustomer']).toBeDefined();
        expect(contracts['getOrder']).toBeDefined();
        expect(contracts['getInvoice']).toBeDefined();

        Object.values(contracts).forEach((schema) => {
          expect(schema.input).toBeDefined();
          expect(schema.output).toBeDefined();
        });
      });

      then('schemas match snapshot', () => {
        expect(contracts).toMatchSnapshot();
      });
    });
  });
});
